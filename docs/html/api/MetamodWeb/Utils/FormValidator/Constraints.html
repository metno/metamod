NAME

MetamodWeb::Utils::FormValidator::Constraints - Data::FormValidator constraint functions that can be used in form profiles.

DESCRIPTION

The module implements "new" style Data::FormValidator constraints. See the documentation of each constraint for details.

All these constraint functions are based on closures to be sure you understand closures before changing them.

FUNCTIONS/METHODS

xml($schema)

Create a constraint function that validates that the value is both valid XML and that it conforms to the supplied schema.

$schema

The path to a XSD schema.

return

Returns a constraint functions that will validate a field according to the schema.
