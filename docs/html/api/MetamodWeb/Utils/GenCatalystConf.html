NAME

MetamodWeb::Utils::GenCatalystConf - Generate a Catalyst configuration from the information found in the a master config.

DESCRIPTION

This module is used to auto-generate a Catalyst configuration file based on the information found in a master config file. It is the intention that the generated Catalyst configuration should be used with out any manually editing afterwards.

FUNCTIONS

catalyst_config()

Generate a Catalyst configuration in JSON format from the information found in the master config file.

return

Returns the Catalyst configuration as a JSON string.
