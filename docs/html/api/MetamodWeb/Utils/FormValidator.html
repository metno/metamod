NAME

MetamodWeb::Utils::FormValidation - Module for performing form validation.

DESCRIPTION

This module provides a thin wrapper around Data::FormValidator to allow attaching labels to each field for better error reporting to the user.

ATTRIBUTES

valid_profile

A hash ref with a Data::FormValidator compatible form profile. In addition to the standard Data::FormValidator attributes it can contain the key 'labels' which takes as hash ref as value. The 'labels' hash ref is a mapping between field names and labels shown to the user.

field_labels

A hash reference of field labels. Set automatically when validation profile is set.

result

The Data::FormValidator::Result object from the last validation.

FUNCTIONS/METHODS

$self->_remove_labels($new_profile, $old_profile)

Trigger function called automatically by Moose when the attribute validation_profile is set.

$new_profile

The new attribute value.

$old_profile

The old attribute value.

$self->validate($params)

Validate the parameters against the validation profile using Data::FormValidator::check()

$params

A hash reference of the form parameters to check.

return

Returns a Data::FormValidator::Result object.

$self->field_label($field_name)

$field_name

The name of field in the form profile.

return

If the form field has a label the label is returned. Otherwise it returns undef.
