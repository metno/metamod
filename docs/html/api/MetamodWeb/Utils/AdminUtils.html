NAME

MetamodWeb::Utils::AdminUtils

DESCRIPTION

Various utility methods for sysadmin interface

METHODS

list_files($dir, $path, $maxfiles)

Counts and lists the files belonging to a dataset. NB: Does not (yet) support multiple hierarchies of directories (like used in OSI SAF).

Returns a ref to an array [$count, \@files], where $count is the total number of files and @files contains the names of the first $maxfiles files (without extensions).

$dir

The path to the xml file directory (metadata)

$path

The dataset name

$maxfiles

Only return this many filenames. If undef, returns all.

read_file($file)

Read the contents of a file and return as string.

$file

The complete pathname to the file

write_file($file, $content)

Write $content data to $file (via tempfile so won't clobber if aborted).

$file

The complete pathname to the file

$content

Data string to be written

$charset

(Optional) Write file using specified encoding (using Perl-specific names, e.g. 'UTF-8')

validate($xml, $schema)

Validate XML string against XML Schema file

$xml

XML string to check

$schema

Full path to XML Schema file
