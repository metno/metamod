NAME

MetamodWeb::Utils::UI::DatasetAdmin - Utility functions for building the dataset admin UI.

FUNCTIONS/METHODS

$self->gcmdlist($quest_element)

Get the gcmd option list for a questionnaire element of type gcmdlist.

$quest_element

The questionnaire element to get the list for. The element is expected to be a hash reference with the required key 'value' which hold the relative filename of the file with all the list elements. The filename should be relative to the base target directory. In addition the keys 'exclude' and 'include' are supported which will remove or add elements respectively.

return

A reference to a list with all the keywords.
