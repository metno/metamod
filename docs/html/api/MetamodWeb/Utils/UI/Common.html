NAME

MetamodWeb::Utils::UI::Common - Utility functions used for building the UI common for most pages

SYNOPSIS

DESCRIPTION

FUNCTIONS/METHODS

$self->app_menu()

return

Returns the APP_MENU configuration variable as a hash reference. The key is the text to display and the value is the link for the item.

$self->stringify_params($params)

Make an URL escaped version of the request parameters. Usefull when you need to keep the current request parameters across several request.

$params

A hash reference with the CGI parameters that you want to stringify. All values will be URI escaped.

return

The stringified version of the parameters.
