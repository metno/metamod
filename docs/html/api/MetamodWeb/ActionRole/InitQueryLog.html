NAME

MetamodWeb::ActionRole::InitQueryLog - A Moose/Catalyst action role used to initialise a query log.

DESCRIPTION

This module implements a Moose/Catalyst action role that when applied to an action with :Does('InitQueryLog') will initialise a DBIx::Class query log.

The DBIx::Class query log is used to a see which SQL queries are actually generated by DBIx::Class. This is very helpful during development and debugging since the same queries that are used by DBIx::Class can also be run in database console and you can see that your DBIx::Class searches generate the expected SQL.

Query logging will only be applied if the environment variable METAMOD_DBIC_TRACE is set to 1.

This action role should be used together with MetamodWeb::ActionRole::DumpQueryLog.
