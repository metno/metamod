NAME

MetamodWeb::Controller::Dataset - Controller for viewing information about a single dataset.

DESCRIPTION

This controller implements action for retrieving information about a single dataset.

METHODS

index

Action for viewing all datasets that are available in the catalog. This page is primarily intended for search engines.

ds_id

Root for chained actions. Used to get the ds_id from the URL and check that it actually matches a dataset in the catalog.

If the dataset cannot be found the request is detached to the 'default' action in the Root controller.

xml

A chained action for displaying the metadata XML for a dataset.

wmsinfo

A chained action for displaying the wmsinfo XML for a dataset.

view

A chained action for viewing the metadata for a dataset in HTML.

wmssetup

A chained action for getting the WMS setup info for a dataset. The WMS setup info is returned as a XML document.

rss

A chained action for generating a RSS feed for a level 1 dataset.
