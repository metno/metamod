NAME

import_dataset.pl - Import an dataset or a directory with datasets into the metadata database

DESCRIPTION

Imports an XML file or a directory of metadata XML files inthe the metadata database.

USAGE

  import_dataset.pl <file or dir> [options]

Options:

--config

Path to application directory or application config file.

--verbose | -v

Set loglevel to INFO

--debug | -d

Set loglevel to DEBUG

--warn

Set loglevel to WARN

--error

Set loglevel to ERROR

--quiet | -q

Set loglevel to OFF

Note that the --config option may come before the filename/directory argument, but the loglevel options must come after or they will eat up the next argument.
