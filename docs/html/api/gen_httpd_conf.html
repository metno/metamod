NAME

gen_httpd_conf.pl - Apache config generator for Metamod

DESCRIPTION

This utility generates a stub Apache config to be placed somewhere in sites-available (if using VirtualHosts) or conf.d (if using path to specify different sites).

The generated file is written to $target/etc/httpd.conf, or stdout if using -p.

USAGE

Running script

 ./common/scripts/gen_httpd_conf.pl [-p] <application_directory>

OPTIONS

Parameters

-p

Prints output to stdout regardless of setting in master_config.

application_directory

'application_directory' is the name of a directory containing the application specific files. Inside this directory, there must be a master_config.txt file.
