NAME

gen_httpd_conf.pl - Proxy config generator for Metamod

DESCRIPTION

This utility generates a stub Apache or nginx config to be placed somewhere in sites-available (if using VirtualHosts) or conf.d (if using path to specify different sites).

The generated file is written to $target/etc/httpd.conf (for Apache), $target/etc/nginx.conf for nginx, or stdout if using -p.

USAGE

Running script

 ./common/scripts/gen_httpd_conf.pl [-p] [--nginx] [--config <config file or dir>]

OPTIONS

Parameters

-p

Prints output to stdout regardless of setting in master_config.

--config

Directory containing master_config.txt and where to write generated ./etc files.
