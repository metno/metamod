NAME

Catalyst::DispatchType::Path - Path DispatchType

SYNOPSIS

See Catalyst::DispatchType.

DESCRIPTION

Dispatch type managing full path matching behaviour. For more information on dispatch types, see:

  • Catalyst::Manual::Intro for how they affect application authors
  • Catalyst::DispatchType for implementation information.

METHODS

$self->list($c)

Debug output for Path dispatch points

$self->match( $c, $path )

For each action registered to this exact path, offers the action a chance to match the path (in the order in which they were registered). Succeeds on the first action that matches, if any; if not, returns 0.

$self->register( $c, $action )

Calls register_path for every Path attribute for the given $action.

$self->register_path($c, $path, $action)

Registers an action at a given path.

$self->uri_for_action($action, $captures)

get a URI part for an action; always returns undef is $captures is set since Path actions don't have captures
