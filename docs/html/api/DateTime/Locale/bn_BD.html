NAME

DateTime::Locale::bn_BD

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'bn_BD' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Bengali Bangladesh.
