NAME

DateTime::Locale::pt

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'pt' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Portuguese.

DATA

This locale inherits from the DateTime::Locale::root locale.
