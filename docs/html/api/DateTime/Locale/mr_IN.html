NAME

DateTime::Locale::mr_IN

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'mr_IN' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Marathi India.

DATA

This locale inherits from the DateTime::Locale::mr locale.

It contains the following data.
