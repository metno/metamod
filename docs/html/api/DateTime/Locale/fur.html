NAME

DateTime::Locale::fur

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'fur' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Friulian.

DATA

This locale inherits from the DateTime::Locale::root locale.
