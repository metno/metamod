NAME

DateTime::Locale::te_IN

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'te_IN' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Telugu India.

DATA

This locale inherits from the DateTime::Locale::te locale.

It contains the following data.
