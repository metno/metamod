NAME

DateTime::Locale::nl_BE

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'nl_BE' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Dutch Belgium.

DATA

This locale inherits from the DateTime::Locale::nl locale.
