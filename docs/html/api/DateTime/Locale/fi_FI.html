NAME

DateTime::Locale::fi_FI

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'fi_FI' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Finnish Finland.

DATA

This locale inherits from the DateTime::Locale::fi locale.
