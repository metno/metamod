NAME

DateTime::Locale::ti_ET

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'ti_ET' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Tigrinya Ethiopia.
