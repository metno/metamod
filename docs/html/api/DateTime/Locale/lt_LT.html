NAME

DateTime::Locale::lt_LT

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'lt_LT' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Lithuanian Lithuania.
