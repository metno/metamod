NAME

DateTime::Locale::hr

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'hr' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Croatian.

DATA

This locale inherits from the DateTime::Locale::root locale.
