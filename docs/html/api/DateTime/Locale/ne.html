NAME

DateTime::Locale::ne

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'ne' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Nepali.

DATA

This locale inherits from the DateTime::Locale::root locale.
