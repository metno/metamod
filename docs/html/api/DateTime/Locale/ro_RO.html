NAME

DateTime::Locale::ro_RO

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'ro_RO' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Romanian Romania.

DATA

This locale inherits from the DateTime::Locale::ro locale.
