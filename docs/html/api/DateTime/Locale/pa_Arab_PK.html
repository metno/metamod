NAME

DateTime::Locale::pa_Arab_PK

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'pa_Arab_PK' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Punjabi Arabic Pakistan.

This locale inherits from the DateTime::Locale::pa_Arab locale.
