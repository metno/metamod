NAME

DateTime::Locale::tg_Cyrl_TJ

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'tg_Cyrl_TJ' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Tajik Cyrillic Tajikistan.

This locale inherits from the DateTime::Locale::tg_Cyrl locale.
