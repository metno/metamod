NAME

DateTime::Locale::so_KE

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'so_KE' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Somali Kenya.

This locale inherits from the DateTime::Locale::so locale.
