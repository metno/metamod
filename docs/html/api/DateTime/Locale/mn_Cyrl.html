NAME

DateTime::Locale::mn_Cyrl

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'mn_Cyrl' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Mongolian Cyrillic.
