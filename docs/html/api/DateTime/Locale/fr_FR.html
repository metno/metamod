NAME

DateTime::Locale::fr_FR

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'fr_FR' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for French France.

This locale inherits from the DateTime::Locale::fr locale.
