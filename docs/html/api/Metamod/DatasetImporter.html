NAME

Metamod::DatasetImporter - Module for importing dataset information into the metadata database.

SYNOPSIS

    use Metamod::DatasetImporter;
    my $importer = Metamod::DatasetImporter->new();
    eval { $importer->write_to_database($inputfile); } or ...

DESCRIPTION

Parses XML files in various formats (ISO 19115, DIF plus met.no formats MM2 and MMD) and inserts metadata in database. Formats are supported via plugins in the Metamod::DatasetTransformer namespace.

METHODS

$self->write_to_database($inputBaseFile)

Write information about the database set to the index database.

$inputBaseFile

The file name for the metadata file. The filename can be both with and without the .xml and .xmd extension.

return

Returns 1 on success. Dies on failure, so must be eval-ed.

$self->_remove_old_metadata($dsid)

Remove metadata that is stored in the metabase and associated with a dsid.

$dsid

The dsid of the dataste.

return

Always 1. Throws and exception on database error.

$self->_get_dsid($ds)

Get the <dsid> for the dataset or a new dsid in case of first time import.

$ds

A reference to the dataset object.

return

Returns either the current dsid for the dataset or a new dsid for new datasets.

$self->_get_parent_id($ds)

Get the dsid of the parent dataset if the dataset has a parent.

$ds

A dataset object

return

Returns the dsid if the parent dataset if the dataset has a parent. Returns 0 if the database does not have a parent.

$self->_transform_metadata($ds)

Transform the dataset metadata to a format understood by the database.

$ds

A reference to a Metamod::ForeignDataset

return

An array where all the values are tuples of the type [ <metadata name>, <value>]
