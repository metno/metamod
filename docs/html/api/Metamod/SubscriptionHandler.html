NAME

Metamod::SubscriptionHandler - Base class for all subscription handlers

DESCRIPTION

This module is a base class for all subscription handlers.

FUNCTIONS/METHODS

new()

$self->push_to_subscribers( $ds, $subscribers )

Push the new file to the subscripers.

$ds

A reference to Metamod::Dataset object

$subscribers

A reference to list of subscriptions. Each subscription is a hash reference with the information neccessary for the specific subscription handler.

return

Returns 1 on success. False otherwise.

$self->_get_userbase()

return

Returns a Metamod::mmUserbase object if one can be created. If the object cannot be created it returns undef.
