NAME

Metamod::DatasetTransformer::OldDataset - transform old-dataset to dataset and MM2

SYNOPSIS

  use Metamod::DatasetTransfomer::OldDataset.pm;
  my $dsT = new Metamod::DatasetTransfomer::Dataset($xmdStr, $xmlStr);
  my $datasetStr;
  if ($dsT->test) {
      $ds2Obj = $dsT->transform;
  }

DESCRIPTION

This module is an implentation of Metamod::DatasetTransformer to convert the old dataset format to the dataset and MM2 format. Old datasets combined the metadata-information and the metadata of metadata (now called dataset) into one file. There doesn't exist a .xmd file for the old datasets.

METHODS

For inherited options see Metamod::DatasetTransformer, only differences are mentioned here.

new($xmlStr, %options)

Options include:

dsXslt => 'filename.xslt'

Overwrite the default xslt file to convert to the actual dataset format.

mm2Xslt => 'filename.xslt'

Overwrite the default xslt file to convert to the mm2 format.
