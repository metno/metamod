NAME

Metamod::DatasetTransformer::MMD - transform MMD to MM2

DESCRIPTION

This module is an implentation of Metamod::DatasetTransformer to convert the MMD format to the dataset (xmd) and MM2 format. MMD files may contain information from MM2 and Dataset, but Dataset-information may also be provided through the xmdString.

METHODS

For inherited options see Metamod::DatasetTransformer, only differences are mentioned here.

new($xmdStr, $xmlStr, %options)

Initialize the transformation by the meta-metadata (optional) and the MMD document. Both arguments can be strings or XML::LibXML::Documents.

Options include:

xmdXslt => 'filename.xslt'

Overwrite the default xslt file to convert to the actual dataset format.

mm2Xslt => 'filename.xslt'

Overwrite the default xslt file to convert from dif to the mm2 format.
