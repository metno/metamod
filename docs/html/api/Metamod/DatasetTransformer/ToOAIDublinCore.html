NAME

Metamod::DatasetTransformer::ToOAIDublinCore - transform foreign datasets to OAI Dublin Core format.

SYNOPSIS

  use Metamod::DatasetTransformer::ToAOIDublinCore qw(foreignDataset2oai_dc);
  my $foreignDataset = ...;
  my $dc;
  eval { $dc = foreignDataset2oai_dc($foreignDataset) };
  if ($@) { ... };

DESCRIPTION

The Metamod::DatasetTransformer::ToOAIDublinCore reads (DIF, ISO, MM2, OldDataset) file formats and converts them to OAI Dublin core format by first transforming the dataset to MM2 if necessary and then doing a transformation from MM2 to OAI Dublic Core.

Please note that having metadata in OAI Dublin Core from the start is not supported.

FUNCTIONS

foreignDataset2oai_dc($foreignDataset)

Return: $foreignDataset (in DIF format) Throws: on file-system related problems if the xml-schema doesn't match if no parameter given
