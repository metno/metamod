NAME

Metamod::DatasetTransformer::MM2 - identity transform of MM2

SYNOPSIS

  use Metamod::DatasetTransfomer::Dataset2.pm;
  my $dsT = new Metamod::DatasetTransfomer::Dataset2($xmlStr);
  my $datasetStr;
  if ($dsT->test) {
      $ds2Obj = $dsT->transform;
  }

DESCRIPTION

This module is an implentation of Metamod::DatasetTransformer. It does the identity conversion of the dataset2 format.

METHODS

For inherited options see Metamod::DatasetTransformer, only differences are mentioned here.

new($xmdStr, $xmlStr, [%options])

In addition to the default initialization by strings, this implementation also allows initialization by XML::LibXML::Nodes.
