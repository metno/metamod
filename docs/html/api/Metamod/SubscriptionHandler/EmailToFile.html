Metamod::SubscriptionHandler::EmailToFile - Subscription handler that sends email to file. Used for automatic testing.

DESCRIPTION

This class is used to override the neccessary parts of Metamod::SubscriptionHandler::Email so that emails are written to a file instead of being being actually sent. This simplifies testing.
